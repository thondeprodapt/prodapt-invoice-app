# PRODAPT Invoice Studio

Open `index.html` to use the app on this computer.

For iPhone use:

1. Upload this folder to a simple HTTPS web host.
2. Open the hosted `index.html` link in Safari on the iPhone.
3. Tap Share, then Add to Home Screen.
4. Add your company logo from Settings. It appears in the document header and as a watermark on quotations and invoices.
5. Use Export PDF to open the print/PDF flow, then share the PDF by Mail, Gmail, WhatsApp, or Files.

Data is saved in the browser on the device using local storage. Keep a backup if you clear Safari/browser data.
